A simple plugin which helps you to find your widget in your Widgets page in WordPress Admin

== Description ==

Do you have lots of widgets installed? Are you tired of finding your required widget from the large number of widgets. This plugin helps you to find your required widget by keyword filter.

== Installation ==

= Using The WordPress Dashboard =

1. Navigate to the 'Add New' Plugin Dashboard
1. Select 'easy-widget-finder.zip' from your computer
1. Install
1. Activate the plugin on the WordPress Plugin Dashboard

= Using FTP =

1. Extract 'easy-widget-finder.zip' to your computer
1. Upload the 'easy-widget-finder' directory to your '/wp-content/plugins/' directory
1. Activate the plugin on the WordPress Plugins dashboard
